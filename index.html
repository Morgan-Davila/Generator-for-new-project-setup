Créer une page HTML vierge

Entrez les données du head de la page que vous voulez créer puis appuyez sur le bouton. Celui-ci vous donnera un fichier .zip avec tout ce qu'il faut pour votre projet web.

Les métadonnées

Le CSS

(.css sera ajouté automatiquement)

Le JavaScript

(.js sera ajouté automatiquement)

Les dossiers, pour bien organiser votre projet

Les dossiers suivants seront déjà pré-installés :

  • Un dossier "style" (dans lequel seront mis les fichiers CSS)
  • Un dossier "js" (dans lequel seront mis vos fichiers JS)
  • Un dossier "assets" (pour les photos, vidéos et autres fichiers)
  • Un dossier "font" avec des polices pré-installées
